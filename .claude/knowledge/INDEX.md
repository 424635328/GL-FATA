## Terms

- **GL-FATA**: this repository's modified optimizer.
- **FEs**: function evaluations; the comparison budget.
- **Run0529**: retained eight-algorithm CEC2022 output snapshot under `CEC2022/RunCEC2022/Run0529/`.
